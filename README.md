# DoseBench
Benchmark and evaluation framework for reliability analysis of OTC medication reasoning in LLMs.

Code, prompts, benchmark scenarios, and evaluation framework for:
"Can I Take Another Dose? Evaluating LLM Decision-Making Under Temporal Uncertainty in OTC Dosing*"

📄 Paper: https://arxiv.org/abs/2606.04262

Repository currently under active organization and will be updated progressively.
